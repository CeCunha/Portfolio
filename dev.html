Burger Factory

Web page to simulate a burger creation of restaurant app. Repository based on lab/exercicio js course (Object). Resources used: JS, HTML, CSS.



THINK! - Wordle Game

Game created using JS, HTML and CSS programming skills. Use of bootstrap and API for responsiveness and functionality. It has gameplay in English and Portuguese (Brazil) languages.


Inspired by the web game Wordle (Josh Wardle), with some adaptations, the player must guess the mysterious 6-letter word in 5 attempts. Think! and good luck !


Save The Boy - Guessing Game

The Guessing Game is a game created from a lab exercise focusing on JS, HTML and CSS. For formatting the layout, bootstrap was used.


The game is a guessing game where the user is asked to guess a number between 1 and 20. If the user guesses correctly, they win. If they guess incorrectly, they lose. The game ends when the user guesses correctly or incorrectly 3 times.


Crazy Chef Toy

Web page to simulate interactive children's book with buttons and sounds. Project for Assignment of JS, using event listener, loops, DOM manipulation. CSS and Bootstrap for layout and responsiveness HTML.


On this interactive page, it's possible simulate children's audio books. You can hear the names of the characters when pressing the button, in addition to generating a random story or creating your own story


Weather Controller

Page created to simulate climate control. Lab based JS programming course covering: JS - Array, If, Do While Loop and Switch. CSS and Bootstrap for layout and responsiveness HTML - Select Option elements


On this interactive page, you can simulate the weather control, as well as the time of day, add elements such as rain, wind and clouds, and also start simulating a timelapse.
